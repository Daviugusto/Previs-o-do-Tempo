App de Previsão do Tempo

Aplicação web que consulta dados meteorológicos em tempo real por meio de uma API, permitindo visualizar as condições climáticas de qualquer cidade.

🚀 Funcionalidades
Pesquisa por cidade
Temperatura atual
Umidade do ar
Condições climáticas
Ícone do clima
Interface responsiva
🛠️ Tecnologias
HTML5
CSS3
JavaScript
API OpenWeather
🎯 Objetivo

Praticar o consumo de APIs REST utilizando JavaScript, manipulação do DOM e exibição dinâmica de informações em uma interface amigável.
